# Elias Galindo — Portfolio

Personal portfolio website built with pure HTML, CSS, and JavaScript. No frameworks, no build tools — open `index.html` directly in any browser.

## Live Preview

Open `index.html` in your browser or host it on any static file server.

## Features

- Dark theme with glassmorphism project cards
- Typing animation cycling through specializations
- Scroll-reveal animations via Intersection Observer
- Animated skill bars
- Mouse-tracking glow effect on project cards
- Fully responsive (mobile, tablet, desktop)

## Structure

```
portfolio/
├── index.html   # Single-page structure (nav, hero, about, skills, projects, contact)
├── style.css    # Dark theme, CSS variables, animations, responsive grid
├── script.js    # Typing effect, scroll reveals, skill bar animation
└── README.md
```

## Sections

- **Hero** — Name, title, typing animation, GitHub & LinkedIn links
- **About** — Bio and stats (languages, repos, experience, location)
- **Skills** — Grouped by category with animated progress bars
  - Systems & Backend: Rust, C++, C#, Go, PHP
  - Web & Frontend: JavaScript, TypeScript, React, Node.js, Ionic
  - Mobile: Flutter / Dart
  - Scripting & Data: Python
- **Projects** — 9 featured GitHub repositories with language badges
- **Contact** — GitHub and LinkedIn links

## Tech Stack

| Layer | Technology |
|-------|------------|
| Markup | HTML5 |
| Styles | CSS3 (custom properties, grid, flexbox, backdrop-filter) |
| Scripts | Vanilla JavaScript (ES6+) |
| Fonts | Inter + JetBrains Mono (Google Fonts) |
